# CertificateForge — Генератор Сертифікатів

Застосунок для автоматичного генерування PDF-сертифікатів на основі шаблону та списку імен з Excel-файлу.

---

## Швидкий старт (без Python)

1. Завантаж **CertificateForge.zip** зі сторінки [Releases](../../releases/latest)
2. Розпакуй архів у будь-яку папку
3. Запусти **CertificateForge.exe**

---

## Вимоги

- Windows 10 або 11
- [Python 3.10+](https://www.python.org/downloads/) — при встановленні обов'язково постав галочку **"Add Python to PATH"**
- Microsoft Edge WebView2 Runtime — на більшості Windows 10/11 вже встановлений. Якщо ні — [завантажити тут](https://developer.microsoft.com/uk-ua/microsoft-edge/webview2/)

---

## Встановлення

Відкрий папку з програмою, натисни правою кнопкою миші на порожньому місці → **"Відкрити в терміналі"** (або **PowerShell**), і виконай по черзі:

**1. Створи віртуальне середовище:**
```
python -m venv .venv
```

**2. Активуй його:**
```
.venv\Scripts\activate
```

**3. Встанови залежності:**
```
pip install -r requirements.txt
```

> Це може зайняти 2–5 хвилин, зачекай до завершення.

---

## Запуск

Після встановлення кожного разу запускай так:

**1. Активуй середовище (якщо ще не активоване):**
```
.venv\Scripts\activate
```

**2. Запусти програму:**
```
python main.py
```

Відкриється вікно програми.

---

## Як користуватися

1. **Завантаж PDF-шаблон** — натисни відповідну кнопку та обери файл шаблону сертифіката.
2. **Завантаж Excel-файл** — перший стовпець має містити список імен (ПІБ) учасників.
3. **Налаштуй позицію тексту** — перетягни мітку імені на потрібне місце поверх прев'ю сертифіката.
4. **Натисни "Генерувати"** — обери папку для збереження. Програма створить окремий PDF-сертифікат для кожного імені зі списку.

## Можливі проблеми

**Помилка `'python' is not recognized`**
→ Python не доданий до PATH. Перевстанови Python з галочкою "Add Python to PATH".

**Вікно не відкривається / помилка WebView2**
→ Встанови [Microsoft Edge WebView2 Runtime](https://developer.microsoft.com/uk-ua/microsoft-edge/webview2/).

**Помилка під час `pip install`**
→ Переконайся, що активоване середовище (у терміналі має бути `(.venv)` на початку рядка).
